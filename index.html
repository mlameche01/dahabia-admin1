Créer le card product dzd_virtual_card
Via curl (terminal) : créer le card product avec payment_instrument = VIRTUAL_PAN. Token = dzd_virtual_card.
curl -u "APP_TOKEN:ADMIN_TOKEN" -X POST \
https://sandbox-api.marqeta.com/v3/cardproducts \
-H "Content-Type: application/json" \
-d '{"token":"dzd_virtual_card","name":"DZD Virtual Card","active":true,
"config":{"fulfillment":{"payment_instrument":"VIRTUAL_PAN"},
"poi":{"ecommerce":true},"card_life_cycle":{"activate_upon_issue":true}}}'
